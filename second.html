Question-2: What is the difference between innerHTML, innerText, and textContent ?

Answer-2:

innerHTML gives the full content with html tags.
innerText only gives the text/content without html tags but it also shows the styles or design.
textContent also gives the text/content without html tags but it doesn't show the styles or design.It gives row content.

Question-3: What is event delegation in the DOM?

Answer-3:

Event delegation is a technique where adding a single event lister to a parent element can handle it's child element event dynamically.

Question-4: What is event bubbling in the DOM?

Answer-4:

Event bubbling works in DOM using node. It starts from the target element then it's bubble up to it's parent node until reaching the root node.

Question-5: How do you create, add, and remove elements using JavaScript?

Answer-5:

Create : using createElement() method can create a new tag.
Add : using appendChild() method can add a new element.
Remove : using removeChild() method can remove an element.
